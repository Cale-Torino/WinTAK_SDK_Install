Detailed Description

An ARenderable for drawing Falconview-style axes

Applies visual styles to instances of IVectorShape

Class that can be used to display text based information on the screen.

A collection of layers that make up a scene

Ribbons are used to give Primitives a two dimensional line style (a zig-zag line, for example) without adding excessive amounts of geometry to the scene

Class that displays a rectangle on the map.

Class that displays primitives on the map, defined in pixel coordinate space

The PrimitiveBuilder builds Primitives, as the name implies

The most basic graphics unit in Spyglass

Label that is orientated based off of its ancor and orientating positions.

An offscreen indicator for MapMarkers

Represents an Octree node, used for quickly deciding what needs to be rendered

Loads Android *.9.png files and displays them at arbitrary sizes

DEPRECATED

Class that is designed to bring user's attention to the map by placing a flashing marker on the map.

Represents a collection of related ARenderables that make up a logical layer

Immutable representation of a renderable Icon.

Class intended for use by the Font. Do not try to directly use this unless absolutely necessary.

Renders a fan or sector

Class that displays an ellipse on the map.
