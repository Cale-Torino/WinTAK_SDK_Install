Detailed Description

This small interface should be used when a map object has a non-standard method of determining the title that should be displayed in various places around WinTAK. Normally, the /event/detail/contact/@callsign value is used from a map object's source for it's title in the Overlay Manager or the MapObjectInfo pane. This object will override this title when added to the MapObject's Properties using the key "map-object-title".
