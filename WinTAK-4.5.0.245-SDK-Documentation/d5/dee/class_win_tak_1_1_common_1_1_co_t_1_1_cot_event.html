WinTak.Common.CoT.CotEvent.CotEvent ( )

Create a default state event. The default state is missing the required attributes 'uid', 'type', 'time', 'start', and 'stale'; the required attributes must be set before the event is valid (IsValid()). The required inner point tag is default to 0,0 and no 'detail' tag exists.
