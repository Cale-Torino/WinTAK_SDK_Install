About

The Windows Tactical Assault Kit (WinTAK) is a Government-off-the-Shelf (GOTS) software application and mapping framework for Windows devices. WinTAK has been designed and developed to run on Windows operating system both in a tactical environment and in a Command and Control environment. The WinTAK software application is an extensible moving map display that integrates Department of Defense (DoD) and commercial imagery, map and overlay information to provide enhanced collaboration and Situational Awareness (SA) over a tactical meshed network. WinTAK promotes information flow and communications from the tactical envrionment to command enterprise locations.

Plugins

WinTAK offers the ability to extend the application using buttons on the ribbon bar and new dockable windows. WinTAK utilizes the Managed Extensibility Framework (MEF) to add functionality. For additional documentation on extending WinTAK please reference the Extending WinTAK guide in the WinTAK Sample Plugin Repo.

Frequently Used Interfaces

Below is a list of Interfaces that are frequently used by Plugin Developers.

  • IDockingManager Used to obtain a reference to a dockable window pane by it's ID and activate it
  • IMessageHub Used to send/receive messages
  • ICommunicationService Used to send cot messages and mission packages to other users
  • ILocationService Used to get the current GPS position, speed, heading
  • IMapObjectFinderService Used to get a map object from it's unique ID
  • IMapObjectRenderer Interface responsible for rendering most CoT markers
  • ICotMessageReceiver Interface used to listen for all CoT message traffic
  • ICotMessageSender Interface used to send CoT message updates
  • IGeocoderService Interface used for Geocoding and Reverse Geocoding
