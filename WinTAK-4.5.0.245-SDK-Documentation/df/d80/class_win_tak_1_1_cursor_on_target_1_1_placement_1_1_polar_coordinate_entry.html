Detailed Description

View model for Polar Coordinate Entry menu. This menu can be used to create a new point based on an existing point and a given range and bearing. Also provides options for panning the map to the newly created point ("Slew To") as well as zooming the current map view to a zoom level that will show both the newly created point and the original point used as a reference ("Auto Zoom").
