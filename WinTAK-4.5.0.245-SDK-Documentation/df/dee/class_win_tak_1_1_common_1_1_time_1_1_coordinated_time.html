Detailed Description

A date-time value as per the Cursor on Target specification. This class ensures proper parsing and string exporting are properly formated.

Constructor & Destructor Documentation

◆ CoordinatedTime() [1/2]

WinTak.Common.Time.CoordinatedTime.CoordinatedTime ( )

For creating new timestamps within the system, this method should be used directly.

Create a CoT time that represents the current time: 1) based on system time measured as the number of milliseconds since Jan. 1, 1970, midnight GMT. 2) or if a GPS delta has been set, the system time corrected to GPS time note: GPS time is not used directly, a correction is used instead, due to the poor granularity of GPS time.
